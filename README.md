# 🎬 CRUD Películas Docker

Proyecto CRUD de películas hecho con Node.js, Express, MySQL y Docker.  
Permite crear, listar, editar y eliminar películas desde una API y una interfaz web simple.
# 🚀 Tecnologías usadas
- Node.js
- Express
- MySQL 8
- Docker
- Docker Compose
- HTML + JavaScript (frontend simple)
# 🧠 Funcionalidades
✔ Crear películas  
✔ Listar películas  
✔ Editar películas  
✔ Eliminar películas  
✔ API REST funcional  
✔ Base de datos en Docker  
✔ Inicialización automática con SQL
